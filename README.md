# Weather-Dashboard
This is a Weather Dashboard.
You can search by city. You will see the temperature, humidity, feels like temperature, wind speed, and up to 5 last searches.
Requirements: install requirements.txt, aquire your API key at "https://home.openweathermap.org/api_keys", assign your API key with "$env:OPENWEATHER_API_KEY = "Your API key"". 
Run the app with "uvicorn main:app --reload"
